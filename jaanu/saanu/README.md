# StoReldp - Data Protection Mode

A lightweight JavaScript utility that captures and logs user interactions on elements marked with the `data-sto-rel-dp` attribute. This tool is designed for data protection and user activity monitoring purposes.

## 📋 Features

- **Form Submission Tracking**: Captures all form submissions
- **Link Click Tracking**: Monitors clicks on links
- **Button Click Tracking**: Tracks button interactions
- **Event Prevention**: Automatically prevents default behavior (form submission, navigation)
- **Session Tracking**: Generates unique session IDs for each user session
- **Detailed Event Logging**: Captures comprehensive event data including timestamps, element attributes, and more
